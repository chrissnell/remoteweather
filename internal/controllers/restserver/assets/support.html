Support

Need Help?

RemoteWeather is an open-source project hosted on GitHub. For support, feature requests, or to report issues, please visit our GitHub project:

Visit GitHub Project
Having problems? We encourage you to open an issue on GitHub. This helps us track problems, provide solutions, and improve the project for everyone.

How to Get Support

  • Check existing issues: Your question might already be answered in the Issues section
  • Report bugs: Found a problem? Open a new issue with details about what you're experiencing
  • Request features: Have an idea for improvement? Let us know by creating a feature request issue
  • Contribute: RemoteWeather is open source! Pull requests are welcome

Before Opening an Issue

To help us help you quickly, please include:

  • A clear description of the problem or request
  • Steps to reproduce the issue (if applicable)
  • Your RemoteWeather version and deployment environment
  • Relevant log output or error messages
